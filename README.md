# 智能停车场管理系统

一个功能完整、架构清晰的Python智能停车场管理系统，支持车辆管理、智能车位分配、用户管理、动态收费计算和多维度报表统计。

## 功能特点

### 🚗 车辆管理
- 车辆注册与信息管理
- 车辆进场/出场记录
- 车牌号码验证
- 车辆状态实时监控

### 🅿️ 智能车位管理
- 100个初始车位自动创建
- 基于车辆类型的智能分配算法
- 车位状态实时更新
- 支持车位预订功能

### 👤 用户管理
- 多角色权限管理（管理员/普通用户）
- SHA256密码加密
- 完整的会员系统
- 余额管理与充值

### 💸 灵活收费
- 可配置的收费规则
- 支持免费时长设置
- 每日最高费用限制
- 会员折扣支持
- 多种支付方式

### 📊 报表统计
- 日报、周报、月报、年报
- 多维度数据统计
- 可视化报表生成
- CSV导出功能

### ⚙️ 系统管理
- 系统配置管理
- 详细日志记录
- 数据库备份与恢复
- 设备管理
- 系统状态监控

## 技术架构

### 后端
- **开发语言**: Python 3.7+
- **数据库**: SQLite（轻量级，无需额外安装）
- **架构模式**: 模块化设计

### 前端
- **GUI框架**: Tkinter（跨平台图形界面）
- **界面风格**: 现代、直观
- **响应式设计**: 支持窗口大小调整

## 快速开始

### 系统要求
- Python 3.7 或更高版本
- tkinter（Python标准库，通常已预装）
- SQLite3（Python标准库）

### 安装步骤

1. **克隆代码库**
   ```bash
   git clone https://github.com/dwDANWebster/parking_system.git
   cd parking_system
   ```

2. **直接运行**（无需安装依赖）
   ```bash
   python main.py
   ```

3. **使用默认账号登录**
   - 用户名: `admin`
   - 密码: `admin123`

## 使用指南

### 1. 车辆进场
1. 点击"车辆管理"标签页
2. 点击"车辆进场"按钮
3. 输入车牌号码
4. 系统自动分配合适车位
5. 确认进场

### 2. 车辆出场
1. 点击"车辆管理"标签页
2. 点击"车辆出场"按钮
3. 输入车牌号码
4. 系统自动计算费用
5. 确认出场并支付

### 3. 管理用户
1. 点击"用户管理"标签页
2. 可以添加、编辑或删除用户
3. 支持修改密码和会员管理

### 4. 生成报表
1. 点击"报表统计"标签页
2. 选择报表类型（日报/周报/月报/年报）
3. 点击"生成报表"
4. 查看报表内容
5. 点击"导出报表"保存为CSV

### 5. 系统配置
1. 点击"系统"菜单
2. 选择"系统配置"
3. 调整系统参数
4. 点击"保存配置"

## 系统配置

### 收费规则配置
- 可配置不同车辆类型的收费标准
- 设置免费时长
- 设定每日最高费用

### 会员等级
| 等级 | 折扣率 | 月费 | 权益 |
|------|--------|------|------|
| 普通会员 | 9折 | 50元 | 基础会员权益 |
| 银卡会员 | 8折 | 100元 | 高级会员权益 |
| 金卡会员 | 7折 | 200元 | VIP会员权益 |
| 钻石会员 | 6折 | 300元 | 顶级会员权益 |

### 初始数据
- 系统自动创建100个车位（5层，每层20个）
- 每层前5个为大型车车位，其余为小型车车位
- 初始管理员账号已创建

## 故障排查

### 常见问题
1. **无法启动**: 检查Python版本是否符合要求
2. **登录失败**: 确认用户名和密码是否正确
3. **数据库错误**: 检查数据库文件权限
4. **GUI显示异常**: 确保tkinter已正确安装

### 日志查看
- 系统日志保存在数据库中
- 可通过"系统日志"标签页查看
- 支持按级别过滤和导出
